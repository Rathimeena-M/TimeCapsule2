# TimeCapsule

Java Swing + MySQL Desktop Diary Application

## Requirements
- Java JDK 17
- IntelliJ IDEA
- MySQL Server
- MySQL Connector/J

## Steps to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Create MySQL database: timecapsule
4. Import the required tables
5. Update DBConnection.java with your MySQL username and password
6. Run Main.java

## Features
- User Registration
- Login
- Diary Entry
- View Diaries
- MySQL Database Connectivity
